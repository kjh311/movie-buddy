* Watches SCSS, LESS, or  STYLUS files and auto compiles to CSS.
* Watches for changes in HTML, SCSS, LESS, STYLUS, and JS files and auto-reloads the browser.
* Minifies CSS file.
* Finds JS errors.
* Concatonates multiple JS files into 1 file.
* Minifies JS file.
* Allows JQUERY
* Compresses Images.
* Bootstrap Ready
* Angular Ready
